# docsify
```bash
git clone https://github.com/dcomposed/docsify.git && cd docsify
docker-compose up -d
```
put your docs in ./docs

For more instructions refer to [docsify](https://docsify.js.org/)

[Demo via github pages](https://dcomposed.github.io/docsify)
